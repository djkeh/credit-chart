# 개인 신용거래융자 잔고 추이 그래프

2024년 1월부터 현재까지 코스피·코스닥 개인 신용거래융자 잔고를 주간 단위로 시각화하는 도구입니다. 금융투자협회 FreeSIS와 금융감독원 DART 공공 API에서 데이터를 수집하고, 주요 종합금융투자사업자(종투사) 10개사의 자기자본을 기반으로 산출한 법적 한도선·업계 자체 관리선을 함께 표시합니다. 이것으로 우리나라 개인 투자자들이 투자를 위해 얼마나 신용 자금을 빌려다 썼는지 그 추이를 확인하실 수 있습니다. 이 데이터는 매주 월요일 아침에 갱신됩니다.

신용 자금을 빌려주는 데에는 한계가 있습니다. 이 그래프 차트는 종투사 신용 대출을 해줄 수 있는 법적 한도선을 함께 표시합니다. 따라서 현재 신용 잔고가 얼마나 남았는지를 알 수 있습니다.

> ⚠️ 이 프로젝트는 공공 데이터를 활용한 **정보 시각화** 도구입니다. 투자 조언이나 금융 상품 권유를 목적으로 하지 않으며, 차트에 표시되는 한도선은 공개 정보를 바탕으로 한 추정값입니다.

## 📊 차트 구성

| 요소 | 설명 |
|------|------|
| **신용거래융자 잔고** | 코스피+코스닥 합산, 주간 단위 (금요일 기준) |
| **법적 한도선** | 종투사 자기자본 합산 × 100% × 신용융자 배분 비율 (추정) |
| **업계 자체 관리선** | 종투사 자기자본 합산 × 60% × 신용융자 배분 비율 (추정) |
| **이벤트 마커** | 잔고에 영향을 준 주요 시장 이벤트 |

## 🔧 로컬 실행

직접 데이터를 수집하거나 차트를 수정하고 싶다면 로컬에서 실행할 수 있습니다.

### 요구 사항

- Python 3.9 이상

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/djkeh/credit-chart.git
cd credit-chart

# 의존성 설치
pip install plotly requests pandas openpyxl

# 실행
python credit_chart.py
```

실행 후 `index.html` 파일이 생성되므로, 이를 웹브라우저에서 확인합니다.

### DART API 키 설정 (선택)

증권사 자기자본 데이터를 직접 수집하려면 [금융감독원 DART](https://opendart.fss.or.kr)에서 무료로 API 키를 발급받아 환경 변수로 설정하세요. 키를 설정하지 않으면 내장된 추정치(fallback)가 자동으로 사용됩니다.

```bash
export DART_API_KEY="your_api_key_here"
python credit_chart.py
```
